Mi Caja de Herramientas Digital

Estas son las herramientas que uso día a día para crear, automatizar y gestionar proyectos. Cada una la elegí porque resuelve problemas reales y me ayuda a trabajar de forma más eficiente.

Automatización

Para que las tareas repetitivas se hagan solas

n8n
Sólido

Mi favorita para conectar aplicaciones. Creo flujos que mueven datos entre diferentes servicios automáticamente. Como tener un asistente digital trabajando 24/7.

Make (ex Integromat)
Bueno

Perfecta para automatizar tareas visuales. Uso su interfaz tipo "bloques" para crear procesos que antes hacía manualmente. Ideal para principiantes en automatización.

IFTTT
Bueno

La herramienta más simple para automatizar. "Si pasa esto, entonces haz aquello". Perfecta para conectar apps populares sin complicaciones técnicas.

Creación de Sitios Web

Lo esencial para hacer páginas que funcionen bien

HTML
Bueno

La estructura de cualquier página web. Sé crear sitios organizados, accesibles y que funcionen bien en todos los dispositivos.

CSS
Bueno

Para que los sitios se vean bien. Manejo diseño responsive (que se adapte a móviles), animaciones suaves y estilos modernos.

JavaScript
En desarrollo

Para agregar interactividad a las páginas. Puedo hacer formularios que funcionen, efectos visuales básicos y conectar con servicios externos.

Backend Básico

Lo que funciona "detrás de escena"

Node.js
Aprendiendo

Para crear servidores simples y APIs básicas. Estoy aprendiendo a hacer que las páginas web se comuniquen con bases de datos y otros servicios.

MongoDB
Básico

Base de datos para guardar información. Sé crear colecciones simples y hacer consultas básicas para almacenar y recuperar datos.

APIs REST
Básico

Entiendo cómo funcionan las APIs (los "puentes" entre aplicaciones). Puedo consumir APIs existentes y crear endpoints simples.

Inteligencia Artificial

Mi equipo de asistentes digitales

ChatGPT
Experto

Mi herramienta principal para resolver problemas complejos. Sé cómo hacer las preguntas correctas para obtener respuestas útiles y específicas.

Claude
Experto

Prompt Engineering

Organización y Datos

Para mantener todo en orden y encontrarlo fácil

Google Sheets
Sólido

Mi navaja suiza para datos. Creo fórmulas avanzadas, automatizo cálculos y conecto hojas con otras aplicaciones. Más poderoso que Excel básico.

Trello
Bueno

Para organizar proyectos con el método Kanban (columnas de "Por hacer", "Haciendo", "Terminado"). Simple pero muy efectivo para seguimiento de tareas.

Notion
Bueno

Como tener un cerebro digital organizado. Combina notas, bases de datos y documentación en un solo lugar. Perfecto para crear sistemas de conocimiento.
